# Estudo com FastAPI

Este projeto foi desenvolvido com o objetivo de estudar e praticar a criação de APIs utilizando Python e o framework FastAPI.

## Tecnologias utilizadas

- Python
- FastAPI
- Uvicorn

## Objetivo

Aprender conceitos fundamentais de desenvolvimento de APIs, incluindo:
- Criação de endpoints
- Validação de dados
- Estruturação de projetos
- Boas práticas no desenvolvimento com FastAPI
